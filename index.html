Project Abstract — FogSecurity-AI2

FogSecurity-AI2 is an AI-driven integrated fog computing security framework designed to address critical challenges in distributed Internet of Things (IoT) environments. As fog computing extends cloud capabilities closer to edge devices, it introduces new security risks including rogue node infiltration, session disruption during mobility, and reduced service availability.

This project proposes a novel architecture combining three complementary mechanisms: an Authentication Node Table (ANT) for lightweight node verification using cryptographic hashing, a Mobility Tracker Protocol (MTP) to maintain seamless session continuity during handoff events, and an AI-based adaptive trust model that dynamically evaluates fog node behavior.

The system demonstrates how integrating authentication, mobility management, and artificial intelligence within the fog layer can simultaneously improve latency, reliability, and security resilience. FogSecurity-AI2 provides a practical model for intelligent, self-adaptive fog infrastructures capable of defending against emerging cyber threats while sustaining uninterrupted IoT services.

This platform serves as a research-driven prototype illustrating the future of secure, autonomous, and high-availability fog computing ecosystems.
